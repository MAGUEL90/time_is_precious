# ASSET GUIDE — Time is Precious

Last updated: 2026-05-16

## Visual Direction
**Time is Precious** uses small, efficient pixel art with a Mesopotamian-inspired identity.

The style should feel:
- Ancient / early civilization.
- Warm and earthy.
- Simple but readable.
- Functional for a top-down management RPG.

## Asset Priority
Prioritize small assets first, especially 16x16 where possible.

Avoid creating large detailed assets too early because they slow down production and increase scope.

## Recommended Base Sizes
- Character head / small portrait test: 16x16.
- Top-down player sprite: 16x16 or 16x24 if readability requires it.
- Worker/NPC sprite: 16x16 or 16x24.
- Item icons: 16x16 or 32x32 depending on UI readability.
- Resource nodes: 16x16.
- Workstation objects: 16x16 to 32x32.
- House/building stages: start small, expand only when needed.

## Core Style Rules
- Use limited palettes.
- Keep silhouettes readable.
- Prefer strong shape language over tiny details.
- Use consistent outline behavior.
- Avoid mixing modern/fantasy styles unless intentionally approved.
- Keep the Mesopotamian theme visible through clothing, tools, buildings, and material choices.

## Current Prototype Asset Needs
Minimum playable prototype should focus only on assets needed to prove the core loop.

### Characters
- Player basic idle/walk placeholder.
- Worker/NPC basic idle/walk placeholder.
- Migrant/resident variation can be palette/accessory-based for now.

### Resource Nodes
- Clay / mud source.
- Wood source.
- Stone source.
- Grass/reed source.

### Items / Resources
- Raw mud/clay.
- Mudbrick.
- Sun-dried mudbrick.
- Wood.
- Stone.
- Food/basic ration.
- Simple clothes.

### Buildings / Objects
- Storage / chest / warehouse placeholder.
- Workstation for processing mudbrick.
- Job Board.
- Worker Hub.
- Basic house stage 0-3.

### UI Icons
- Time icon.
- Food icon.
- Worker icon.
- Storage icon.
- Work/process icon.
- Basic resource icons.

## Clothing / Worker Identity
Workers should look like early laborers, not modern employees.

Possible visual elements:
- Simple cloth headband.
- Rough linen wrap.
- Short tunic.
- Rope belt.
- Simple sandals.
- Basket/tool carried on back or side.

Avoid over-detailing at 16x16. Use silhouette and color blocking.

## Mesopotamian Visual Cues
Use carefully and consistently:
- Mudbrick structures.
- Reed/grass materials.
- Clay tablets.
- Simple robes/tunics.
- Bronze/copper tool hints.
- Warm sand, clay, ochre, brown, dark outline colors.

## What Not To Prioritize Yet
Do not prioritize:
- Large character portraits.
- Many clothing tiers.
- Complex animations.
- Large building variants.
- Decorative city assets.
- VFX-heavy assets.

These can come later after the core loop is playable.

## Asset Production Rule
Every asset should answer:
1. Does this support the playable prototype?
2. Is it small enough to produce efficiently?
3. Does it match Mesopotamian identity?
4. Does it improve readability?

If not, delay it.
