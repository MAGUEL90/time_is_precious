Version: v0.1  
Rule: main canvas maximum is **16x16 px**. Actual asset size can be smaller.

## Status Flow

```text
IDEA → SKETCH → DRAFT → REVIEW → APPROVED → EXPORTED → IN_GAME
```

## Asset Size Rule

Use both size fields:

| Field | Meaning |
|---|---|
| Canvas Size | total canvas size, usually 16x16 |
| Actual Size | visible pixel area, can be 4x10, 8x11, 14x16, etc. |

If actual size is not measured yet, write:

[ ] Silhouette readable
[ ] Fits Mesopotamian tone

APPROVED / REWORK / REJECT

Notes:
